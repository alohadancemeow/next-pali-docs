(๑๔๒) ปัจจัยที่สำหรับประกอบกับกิริยากิตก์นั้น จัดเป็น ๓ พวก เหมือนปัจจัยแห่งนามกิตก์.

| ๑. กิตปัจจัย        | ๒. กิจจปัจจัย | ๓. กิตกิจจปัจจัย          |
| ------------------- | ------------- | ------------------------- |
| อนฺต, ตวนฺตุ, ตาวี. | อนีย, ตพฺพ.   | มาน, ต, ตูน, ตฺวา, ตฺวาน. |

ในปัจจัยทั้ง ๓ พวกนั้น

| ปัจจัย                             | เป็นเครื่องหมาย |
| ---------------------------------- | --------------- |
| อนฺต, มาน.                         | บอกปัจจุบันนกาล |
| ตวนฺตุ, ตาวี, ต, ตูน, ตูวา, ตฺวาน. | บอกอดีตกาล      |
| อนีย, ตพฺพ.                        | บอกความจำเป็น   |

# กิตปัจจัย

### อนฺต.

| บาลี    | คำแปล              | วิเคราะห์                                     |
| ------- | ------------------ | --------------------------------------------- |
| สุณนฺโต | ฟังอยู่            | สุ ธาตุ ในความฟัง, ณา ปัจจัย สนธิ (๑๙)        |
| กโรนฺโต | ทำอยู่             | กรฺ ธาตุ ในความ ทำ, โอ ปัจจัย สนธิ (๑๙)       |
| กเถนฺโต | กล่าวอยู่          | กถฺ ธาตุ ในความ กล่าว, เอ ปัจจัย สนธิ (๑๙)    |
| สุณนฺตี | ฟังอยู่ (อิตถี.)   | สุ ธาตุ ในความฟัง, ณา ปัจจัย สนธิ , ลง อี     |
| กโรนฺตี | ทำอยู่ (อิตถี.)    | กรฺ ธาตุ ในความ ทำ, โอ ปัจจัย สนธิ , ลง อี    |
| กเถนฺตี | กล่าวอยู่ (อิตถี.) | กถฺ ธาตุ ในความ กล่าว, เอ ปัจจัย สนธิ , ลง อี |

### ตวนฺตุ.

| บาลี     | คำแปล             | วิเคราะห์                                                 |
| -------- | ----------------- | --------------------------------------------------------- |
| สุตวา    | ฟังแล้ว           | สุ ธาตุ ในความฟัง, นาม (๖๔).                              |
| ภุตฺตวา  | กินแล้ว           | ภุชฺ ธาตุ ในความ กิจ, เอาที่สุดธาตุ เป็น ตฺ, นาม (๖๔).    |
| วุสิตวา  | อยู่แล้ว          | วสฺ ธาตุ ในความ อยู่, เอา ว เป็น วุ แล้ว ลง อิ, นาม (๖๔). |
| สุตวตี   | ฟังแล้ว (อิตถี.)  | สุ ธาตุ ในความฟัง.                                        |
| ภุตฺตวตี | กินแล้ว (อิตถี.)  | ภุชฺ ธาตุ ในความ กิจ, เอาที่สุดธาตุ เป็น ตฺ.              |
| วุสิตวตี | อยู่แล้ว (อิตถี.) | วสฺ ธาตุ ในความ อยู่, เอา ว เป็น วุ แล้ว ลง อิ.           |

### ตาวี.

| ปุํ.     | อิตฺ.      | นปุํ.    | คำแปล     |
| -------- | ---------- | -------- | --------- |
| สุตาวี   | สุตาวินี   | สุตาวิ   | ฟังแล้ว.  |
| ภุตฺตาวี | ภุตฺตาวินี | ภุตฺตาวิ | กินแล้ว.  |
| วุสิตาวี | วุสิตาวินี | วุสิตาวิ | อยู่แล้ว. |

กิริยาศัพท์ที่ประกอบด้วยกิตปัจจัยอย่างนี้อยู่หน้ากิริยาอาขยาตโดยมาก พึงเห็นว่าเป็นเหมือนบทวิเสสนะ.

# กิจจปัจจัย

### อนีย.

| บาลี    | คำแปล          | วิเคราะห์                                |
| ------- | -------------- | ---------------------------------------- |
| กรณียํ  | อันเขาพึงทำ    | กรฺ ธาตุ ในความ ทำ, เอา น เป็น ณ.        |
| วจนียํ  | อันเขาพึงกล่าว | วจฺ ธาตุ ในความ กล่าว.                   |
| โภชนียํ | อันเขา พึงกิน  | ภุชฺ ธาตุ ในความ กิน, พฤทธิ์ อุ เป็น โอ. |

ศัพท์ที่ประกอบด้วยปัจจัยนี้ บางศัพท์ก็ใช้เป็นนามกิตก์บ้าง.
อุ. ปณีเตน ขาทนีเยน โภชนีเยน ปริวิสิ. เขาเลี้ยงแล้ว ด้วยของควรเคี้ยว ด้วยของควรบริโภค อันประณีต.

### ตพฺพ.

| บาลี        | คำแปล           | วิเคราะห์                |
| ----------- | --------------- | ------------------------ |
| กตฺตพฺพํ    | อันเขาพึงทำ     | ลบ รฺ ซ้อน ตฺ.           |
| วตฺตพฺพํ    | อัน เขาพึงกล่าว | วทฺ ธาตุ เอา ทฺ เป็น ตฺ. |
| ภุญฺชิตพฺพํ | อันเขา พึงกิน   | ลง อิ อาคม.              |

# กิตกิจจปัจจัย.

### มาน.

มาน ปัจจัยนี้ มีคติเหมือน อนฺต ก็แต่ว่า ถ้าไม่มี ย ปัจจัย ซึ่งลงในกรรม เป็นกัตตุวาจก, ถ้ามี ย ปัจจัยนั้น เป็นกัมมวาจก อย่างนี้

| กัตตุวาจก.           | กัมมวาจก.                    |
| -------------------- | ---------------------------- |
| กุรุมาโน - ทำอยู่.   | กริยมาโน - อันเขาทำอยู่.     |
| ภุญฺชมาโน - กินอยู่. | ภุญฺชิยมาโน - อันเขากินอยู่. |
| วทมาโน - กล่าวอยู่.  | วุจฺจมาโน - อันเขากล่าวอยู่. |

### ต.

#### ธาตุมี มฺ และ นฺ เป็นที่สุด ลบที่สุดธาตุ.

| บาลี | คำแปล         | วิเคราะห์                    |
| ---- | ------------- | ---------------------------- |
| คโต  | ไปแล้ว        | คมฺ ธาตุ ในความ ไป, ความถึง. |
| รโต  | ยินดีแล้ว     | รมฺ ธาตุ ในความ ยินดี.       |
| ขโต  | อันเขาขุดแล้ว | ขนฺ ธาตุ ในความ ขุด.         |
| หโต  | อันเขาฆ่าแล้ว | หนฺ ธาตุ ในความ ฆ่า.         |

#### ธาตุ จฺ, ชฺ, และ ปฺ เป็นที่สุด เอาที่สุดธาตุ เป็น ตฺ.

| บาลี   | คำแปล              | วิเคราะห์                  |
| ------ | ------------------ | -------------------------- |
| สิตฺโต | อันเขารดแล้ว       | สิจฺ ธาตุ ในความ รด.       |
| วิตฺโต | สงัดแล้ว           | วิ+วิจฺ ธาตุ ในความ สงัด.  |
| ภุตฺโต | อันเขากินแล้ว      | ภุชฺ ธาตุ ในความ กิน.      |
| จตฺโต  | อันเขาสละแล้ว      | จชฺ ธาตุ ในความ สละ.       |
| คุตฺโต | อันเขาคุ้มครองแล้ว | คุปฺ ธาตุ ในความ คุ้มครอง. |
| ตตฺโต  | ร้อนแล้ว           | ตปฺ ธาตุ ในความ ร้อน.      |

#### ธาตุมี อา เป็นที่สุดก็ดี ต เป็นกัมมวาจกก็ดี ลง อิ.

| บาลี      | คำแปล                | วิเคราะห์                |
| --------- | -------------------- | ------------------------ |
| ฐิโต      | ยืนแล้ว              | ฐา ธาตุ ในความ ตั้งอยู่. |
| ปีโต      | อันเขาดื่มแล้ว       | ปา ธาตุ ในความ ดื่ม.     |
| อภิชฺฌิโต | อันเขาเพ่งจำเพาะแล้ว | อภิ+ฌา ธาตุ ในความ เพ่ง. |
| ภาสิโต    | อันเขากล่าวแล้ว      | ภาสฺ ธาตุ ในความ กล่าว.  |

-- ธาตุ มี 'อา' เป็นที่สุด เอาที่สุดธาตุเป็น 'อี' บ้าง 'อิ' บ้าง.

#### ธาตุมี ทฺ เป็นที่สุดอยู่หน้า แปลง ต เป็น นฺน แล้วลบ ที่สุดธาตุ.

| บาลี   | คำแปล          | วิเคราะห์                                    |
| ------ | -------------- | -------------------------------------------- |
| ฉนฺโน  | อันเขามุงแล้ว  | ฉทฺ ธาตุ ในความ ปิด.                         |
| สนฺโน  | จมแล้ว         | สทฺ หรือ สิทฺ ธาตุ ในความ จม.                |
| รุนฺโน | ร้องให้แล้ว    | รุทฺ ธาตุ ในความ ร้องไห้.                    |
| ฉินฺโน | อันเขาตัดแล้ว. | ฉิทฺ ธาตุ ในความ ตัด.                        |
| ภินฺโน | แตกแล้ว.       | ภิทฺ ธาตุ ในความ แตก.                        |
| ทินฺโน | อันเขาให้แล้ว. | ทา ธาตุ ในความ ให้. (เอา อา ที่ ทา เป็น อิ). |

#### ธาตุมี รฺ เป็นที่สุดอยู่หน้า แปลง ต เป็น ณฺณ แล้วลบ ที่สุดธาตุ.

| บาลี   | คำแปล     | วิเคราะห์                  |
| ------ | --------- | -------------------------- |
| ชิณฺโณ | แก่แล้ว.  | ชิรฺ ธาตุ ในความ คร่ำคร่า. |
| ติณฺโณ | ข้ามแล้ว. | ตรฺ ธาตุ ในความ ข้าม.      |
| ปุณฺโณ | เต็มแล้ว. | ปูรฺ ธาตุ ในความ เต็ม.     |

#### ธาตุมี สฺ เป็นที่สุดอยู่หน้า แปลง ต เป็น ฏฺฐ แล้วลบ ที่สุดธาตุ.

| บาลี    | คำแปล        | วิเคราะห์                  |
| ------- | ------------ | -------------------------- |
| ตุฏฺโฐ  | ยินดีแล้ว.   | ตุสฺ ธาตุ ในความ ยินดี.    |
| หฏฺโฐ   | ร่าเริงแล้ว. | หสฺ ธาตุ ในความ หัวเราะ.   |
| ปวิฏฺโฐ | เข้าไปแล้ว.  | ป+วิสฺ ธาตุ ในความ เข้าไป. |

#### ธาตุมี ธฺ และ ภฺ เป็นที่สุดอยู่หน้า แปลง ต เป็น ทฺธ แล้ว ลบที่สุดธาตุ.

| บาลี    | คำแปล            | วิเคราะห์                 |
| ------- | ---------------- | ------------------------- |
| พุทโธ   | รู้แล้ว.         | พุธฺ ธาตุ ในความ รู้.     |
| กุทฺโธ  | โกรธแล้ว.        | กุธฺ ธาตุ ในความ โกรธ.    |
| รุทฺโธ  | ปิดแล้ว.         | รุธฺ ธาตุ ในความ กั้น.    |
| ลทฺโธ   | อันเขาได้แล้ว.   | ลภฺ ธาตุ ในความ ได้.      |
| อารทฺโธ | อันเขาปรารภแล้ว. | อา+รภฺ ธาตุ ในความ เริ่ม. |

#### ธาตุมี มฺ เป็นที่สุดอยู่หน้า แปลง ต เป็น นฺต แล้วลบ ที่สุดธาตุ.

| บาลี     | คำแปล       | วิเคราะห์                   |
| -------- | ----------- | --------------------------- |
| ปกฺกนฺโต | หลีกไปแล้ว. | ป+กมฺ ธาตุ ในความ ก้าวไป.   |
| ทนฺโต    | ทรมานแล้ว.  | ทมฺ ธาตุ ในความ ทรมาน.      |
| สนฺโต    | ระงับแล้ว.  | สมฺ ธาตุ ในความ สงบ, ระงับ. |

#### ธาตุมี หฺ เป็นที่สุดอยู่หน้า แปลง ต เป็น ฬฺห แล้วลบ ที่สุดธาตุ.

| บาลี   | คำแปล            | วิเคราะห์             |
| ------ | ---------------- | --------------------- |
| รุฬฺโห | งอกแล้ว.         | รุหฺ ในความ งอก.      |
| มุฬฺโห | หลงแล้ว.         | มุหฺ ธาตุ ในความ หลง. |
| วุฬฺโห | อันน้ำพัดไปแล้ว. | วุหฺ ธาตุ ในความ ลอย. |

-- ศัพท์ที่ประกอบด้วยปัจจัยนี้ บางศัพท์ก็ใช้เป็นนามกิตก์บ้าง เหมือนบทว่า พุทฺโธ เป็นต้น.

### ตูนาทิ

| ตูน.    | ตฺวา.    | ตฺวาน.    | คำแปล    |
| ------- | -------- | --------- | -------- |
| กาตูน.  | กตฺวา.   | กตฺวาน.   | ทำแล้ว.  |
| คนฺตูน. | คนฺตฺวา. | คนฺตฺวาน. | ไปแล้ว.  |
| หนฺตูน. | หนฺตฺวา. | หนฺตฺวาน. | ฆ่าแล้ว. |

#### อุปสัคอยู่หน้า แปลงปัจจัยทั้ง ๓ เป็น ย.

| บาลี    | คำแปล       | วิเคราะห์                 |
| ------- | ----------- | ------------------------- |
| อาทาย   | ถือเอาแล้ว. | อา+ทา ธาตุ ในความ ถือเอา. |
| ปหาย    | ละแล้ว.     | ป+หา ธาตุ ในความ ละ.      |
| นิสฺสาย | อาศัยแล้ว.  | นิ+สี ธาตุ ในความ อาศัย.  |

#### ธาตุมี มฺ เป็นที่สุดอยู่หน้า แปลง ย กับที่สุดธาตุ เป็น มฺม.

| บาลี     | คำแปล          | วิเคราะห์                  |
| -------- | -------------- | -------------------------- |
| อาคมฺม   | มาแล้ว.        | อา+คมฺ ธาตุ ในความ ไป.     |
| นิกฺขมฺม | ออกแล้ว.       | นิ+ขมฺ ธาตุ ในความ ออก.    |
| อภิรมฺม  | ยินดียิ่งแล้ว. | อภิ+รมฺ ธาตุ ในความ ยินดี. |

อีกอย่างหนึ่ง นิกฺขมฺม นิ+กมฺ ธาตุ ในความ ออก, ก้าวออก.

#### ธาตุมี ทฺ เป็นที่สุดอยู่หน้า แปลง ย กับที่สุดธาตุเป็น ชฺช.

| บาลี     | คำแปล         | วิเคราะห์                   |
| -------- | ------------- | --------------------------- |
| อุปฺปชฺช | เกิดขึ้นแล้ว. | อุ+ปทฺ ธาตุ ในความ เกิด.    |
| ปมชฺช    | ประมาทแล้ว.   | ป+มทฺ ธาตุ ในความ ประมาท.   |
| อจฺฉิชฺช | ชิงเอาแล้ว.   | อา+ฉิทฺ ธาตุ ในความ ชิงเอา. |

#### ธาตุมี ธฺ และ ภฺ เป็นที่สุดอยู่หน้า แปลง ย กับที่สุดธาตุ เป็น ทฺธา, พฺภ.

| บาลี   | คำแปล      | วิเคราะห์                |
| ------ | ---------- | ------------------------ |
| วิทฺธา | แทงแล้ว.   | วิธฺ ธาตุ ในความ แทง.    |
| ลทฺธา  | ได้แล้ว.   | ลภฺ ธาตุ ในความ ได้.     |
| อารพฺภ | ปรารภแล้ว. | อา+รภฺ ธาตุ ในความเริ่ม. |

#### ธาตุมี หฺ เป็นที่สุดอยู่หน้า แปลง ย กับที่สุดธาตุเป็น ยฺห.

| บาลี    | คำแปล       | วิเคราะห์                 |
| ------- | ----------- | ------------------------- |
| ปคฺคยฺห | ประคองแล้ว. | ป+คหฺ ธาตุ ในความ ประคอง. |
| สนฺนยฺห | ผูกแล้ว.    | สํ+นหฺ ธาตุ ในความ ผูก.   |
| อารุยฺห | ขึ้นแล้ว.   | อา+รูหฺ ธาตุ ในความ ขึ้น. |

แปลง ตฺวา เป็น สฺวา, ตฺวาน เป็น สฺวาน จำเพาะแต่ ทิสฺ ธาตุ อย่างเดียว.

ปัจจัยทั้ง ๓ ตัวนี้ บอกอดีตกาลโดยมากกว่ากาลอื่น จึงได้จัดไว้ในพวกอดีตกาล,
ที่บอกกาลอื่น จักมีแจ้งในวากยสัมพันธ์ข้างหน้า และศัพท์ที่ประกอบด้วยปัจจัยทั้ง ๓ ตัวนี้ นำหน้ากิริยาอาขยาตเป็นนิตย์.

## คำถามชื่อ สาธนะ ธาตุ ปัจจัย และวาจก

[๑๔๖] คำถามชื่อ สาธนะ ธาตุ ปัจจัย และวาจก ให้ผู้ ศึกษาตอบ.

#### ถามสาธนะ และ ปัจจัย

ทา ธาตุ ในความ ให้. ทายโก ชื่อคน ทานํ ชื่อกิริยา ทานํ ชื่อเจตนา ทานํ ชื่อของ ทานํ ชื่อที่.
ปจฺ ธาตุ ในความ หุง. ปาจโก ชื่อคน ปกฺกํ ชื่อข้าว ปาโก ชื่อหม้อข้าว ปจนํ ชื่อกิริยา ปจนํ ชื่อฟืน.
พนฺธ ธาตุ ในความ ผูก. พนฺธโก ชื่อคนผูก พนฺธนํ ชื่อกิริยา พนฺธนํ ชื่อเรือนจำ พนฺธนํ ชื่อโซ่ พนฺโธ ชื่อคนที่เขาผูก.

| ทา ธาตุ ในความ ให้. | ปจฺ ธาตุ ในความ หุง. | พนฺธ ธาตุ ในความ ผูก.    |
| ------------------- | -------------------- | ------------------------ |
| ทายโก - ชื่อคน      | ปาจโก - ชื่อคน       | พนฺธโก - ชื่อคนผูก       |
| ทานํ - ชื่อกิริยา   | ปกฺกํ - ชื่อข้าว     | พนฺธนํ - ชื่อกิริยา      |
| ทานํ - ชื่อเจตนา    | ปาโก - ชื่อหม้อข้าว  | พนฺธนํ - ชื่อเรือนจำ     |
| ทานํ - ชื่อของ      | ปจนํ - ชื่อกิริยา    | พนฺธนํ - ชื่อโซ่         |
| ทานํ - ชื่อที่      | ปจนํ - ชื่อฟืน.      | พนฺโธ - ชื่อคนที่เขาผูก. |
