# ทิคุสมาส

[ ๙๖ ] กัมมธารยสมาสมีสังขยาอยู่ข้างหน้า ชื่อทิคุสมาสๆ นั้น มี ๒ อย่าง คือ

- สมาหาโร.
- อสมาหาโร.

### สมาหารทิคุ

ทิคุสมาส ที่ท่านรวมนามศัพท์ มีเนื้อความเป็นพหุวจนะทำให้เป็นเอนกจนะ นปุํสกลิงค์ ชื่อสมาหารทิคุ อุ. อย่างนี้

| ตัวอย่าง          | สำเร็จรูปเป็น | คำแปล       |
| ----------------- | ------------- | ----------- |
| ตโย โลกา          | ติโลกํ        | โลก ๓.      |
| จตสฺโส ทิสา       | จตุทฺทิสํ     | ทิศ ๔.      |
| ปญฺจ อินฺทฺริยานิ | ปญฺจินฺทฺริยํ | อินทรีย์ ๕. |

### อสมาหารทิคุ

ทิคุสมาส ที่ท่านไม่ได้ทำอย่างนี้ ชื่ออสมาหารทิคุ อุ. อย่างนี้

| ตัวอย่าง     | สำเร็จรูปเป็น | คำแปล          |
| ------------ | ------------- | -------------- |
| เอโก ปุคฺคโล | เอกปุคฺคโล    | บุคคลผู้เดียว. |
| จตสฺโส ทิสา  | จตุทฺทิสา     | ทิศ ๔ ท.       |
| ปญฺจ พลานิ   | ปญฺจพลานิ     | กำลัง ๕ ท.     |

# ตัปปุริสสมาส

[ ๙๗ ] นานศัพท์มี อํ วิภัตติเป็นต้น ในที่สุด ท่านย่อเข้าด้วยบทเบื้องปลาย ชื่อตัปปุริสสมาสๆ นี้ มี ๖ อย่าง คือ

- ทุติยาตปฺปุริโส
- ตติยาตปฺปุริโส
- จตุตฺถีตปฺปุริโส
- ปญฺจมีตปฺปุริโส
- ฉฏฺฐีตปฺปุริโส
- สตฺตมีตปฺปุริโส

---

### ทุติยาตัปปุริสะ

| ตัวอย่าง        | สำเร็จรูปเป็น  | คำแปล                                      |
| --------------- | -------------- | ------------------------------------------ |
| สุขํ ปตฺโต      | สุขปฺปตฺโต     | [ ปุริโส บุรุษ ] ถึงแล้วซึ้งสุข.           |
| คามํ คโต        | คามคโต         | [ ปุริโส บุรุษ ] ไปแล้วสู่บ้าน.            |
| สพฺพรตฺตึ โสภโณ | สพฺพรตฺติโสภโณ | [ จนฺโท พระจันทร์ ] งามตลอดราตรี ทั้งสิ้น. |

| สลฺเลน วิทฺโธ         | สลฺลวิทฺโธ    | [ ชนฺตุ สัตว์ ] อันลูกศรแทงแล้ว. |

### สมาสที่มี น อยู่หน้า

สมาสที่มี น อยู่หน้า อย่างนี้ เรียกว่า อุภยตัปปุริสะบ้าง กัมมธารยะมี น เป็นบทหน้าบ้าง.

| ตัวอย่าง    | สำเร็จรูปเป็น | คำแปล                               |
| ----------- | ------------- | ----------------------------------- |
| น พฺราหฺมโณ | อพฺราหฺมโณ    | [ อยํ ชโน ชนนี้ ] มิใช่พราหมณ์.     |
| น วสโล      | อวสโล         | [ อยํ ชโน ชนนี้ ] มิใช่คนถ่อย.      |
| น อสฺโส     | อนสฺโส        | [ อยํ สตฺโต สัตว์นี้ ] มิใช่ม้า.    |
| น อริโย     | อนริโย        | [ อยํ ชโน ชนนี้ ] มิใช่พระอริยเจ้า. |

ในสมาสนี้ ถ้าพยัญชนะ อยู่หลัง น. เอา น เป็น อ เหมือน อพฺราหฺมโณ เป็นต้น.
ถ้าสระ อยู่หลัง น. เอา น เป็น อน เหมือน อนสฺโส เป็นต้น.
