[๙๔] นามศัพท์ ตั้งแต่ ๒ ขึ้นไป ท่านย่อเข้าเป็นบทเดียวกัน ชื่อสมาสๆ นี้
บางอย่างก็มีในภาษาของเรา เหมือนคำว่า

- "อย่าทำ ซึ่งบาป" ย่อเข้าสมาสเป็น "อย่าทำบาป"
- "สิ่งนี้ก้าไหล่ด้วยทอง" เป็น "สิ่งนี้ก้าไหล่ด้วยทอง"
- "นายให้รางวัลแก่บ่าว" เป็น "นายให้รางวัลแก่บ่าว"
- "สมุดของใคร" เป็น "สมุดใคร"
- "สัตว์ในน้ำ หรือสัตว์เกิดในน้ำ" เป็น "สัตว์น้ำ"

เป็นต้น.

---

# สมาสว่าโดยกิจ

สมาสในบาลีภาษา ว่าโดยกิจมี ๒ อย่าง คือ

- สมาสที่ท่านลบวิภัตติเสียแล้ว เรียกว่า ลุตฺตมาโส - มีอุทาหรณ์ว่า กฐินทุสฺสํ ผ้าเพื่อกฐิน, ราชธนํ ทรัพย์ของพระราชา เป็นต้น.
- สมาสที่ท่านยังมิได้ลบวิภัตติ เรียกว่า อลุตฺตสมาโส - มีอุทาหรณ์ว่า ทูเรนิทานํ วัตถุ มีนิทานในที่ไกล, อุรสิโลโม พฺราหฺมโณ พราหมณ์มีขนที่อก เป็นต้น.

---

# สมาสว่าโดยชื่อ

สมาสนั้น ว่าโดยชื่อมี ๖ อย่าง คือ

- กมฺมธารโย
- ทิคุ
- ตปฺปุริโส
- ทวนฺทฺโว
- อพฺยยีภาโว
- พหุพฺพิหิ

---

# กัมมธารยสมาส

[๙๕] นามศัพท์ ๒ บท มีวิภัตติและวจนะเป็นอย่างเดียวกัน บทหนึ่งเป็นประธาน คือ เป็นนามนาม,
บทหนึ่งเป็นวิเสสนะ คือ เป็นคุณนาม หรือเป็นคุณนามทั้ง ๒ บท มีบทอื่นเป็นประธาน ที่ท่านย่อเข้าเป็นบทเดียวกัน ชื่อ กัมมธารยสมาสๆ นั้น มี ๖ อย่าง คือ

- วิเสสนปุพฺพปโท
- วิเสสนุตฺตรปโท
- วิเสสโนภยปโท
- วิเสสโนปมปโท
- สมฺภาวนปุพฺพปโท
- อวธารณปุพฺพปโท

---

### วิเสสนบุพพบท

วิเสสนบุพพบท มีบทวิเสสนะอยู่ต้น, บทประธานอยู่ข้างหลัง อย่างนี้

| ตัวอย่าง      | สำเร็จรูปเป็น | คำแปล         |
| ------------- | ------------- | ------------- |
| มหนฺโต ปุริโส | มหาปุริโส     | บุรุษใหญ่.    |
| ขตฺติยา กญฺญา | ขตฺติยกญฺญา   | นางกษัตริย์.  |
| นีลํ อุปฺปลํ  | นีลุปฺปลํ     | ดอกอุบลเขียว. |

ในสมาสนี้ เอามหันตศัพท์ เป็นมหา อย่างนี้ มหาราชา พระราชาผู้ใหญ่, มหาธานี เมืองใหญ่, มหาวนํ ป่าใหญ่ เป็นต้น.

บางสมาสท่านก็ย่อบทวิเสสนะ เหลือไว้แต่อักษรตัวหน้า อย่างนี้

| ตัวอย่าง        | สำเร็จรูปเป็น | คำแปล                  |
| --------------- | ------------- | ---------------------- |
| กุจฺฉิตา ทิฏฺฐิ | กุทิฏฺฐิ      | ทิฏฐิ อันบัณฑิตเกลียด. |
| ปธานํ วจนํ      | ปาวจนํ        | คำเป็นประธาน.          |
| สนฺโต ปุริโส    | สปฺปุริโส     | บุรุษ ระงับแล้ว.       |

แม้ถึงอย่างนี้ ก็เป็นวิเสสนบุพพบทแท้.

### วิเสสนุตตรบท

วิเสสนุตตรบท มีบทวิเสสนะ อยู่หลัง บทประธานอยู่หน้า อย่างนี้

| ตัวอย่าง        | สำเร็จรูปเป็น | คำแปล        |
| --------------- | ------------- | ------------ |
| สตฺโต วิเสโส    | สตฺตวิเสโส    | สัตว์วิเศษ.  |
| นโร วโร         | นรวโร         | นระประเสริฐ. |
| มนุสฺโส ทลิทฺโท | มนุสฺสทลิทฺโท | มนุษย์ขัดสน. |

### วิเสสโนภยบท

วิเสสโนภยบท มีบททั้ง ๒ เป็นวิเสสนะ มีบทอื่นเป็นประธาน อย่างนี้

| ตัวอย่าง         | สำเร็จรูปเป็น | คำแปล                                 |
| ---------------- | ------------- | ------------------------------------- |
| สีตญฺจ สมฏฺฐญฺจ  | สีตสมฏฺฐํ     | [ ฐานํ ที่ ] ทั้งเย็นทั้งเกลี้ยง.     |
| อนฺโธ จ วธิโร จ  | อนฺธวธิโร     | [ ปุริโส บุรุษ ] ทั้งบอดทั้งหนวก.     |
| ขญฺโช จ ขุชฺโช จ | ขญฺชขุชฺโช    | [ ปุริโส บุรุษ ] ทั้งกระจอกทั้ง ค่อม. |

ในสัททนีติว่า สิตฺตํ จ ตํ สมฺมฏฺฐํ ฐานํ ฯ คือทั้งรดน้ำทั้งปัด กวาด ฯ

### วิเสสโนปมบท

วิเสสโนปมบท มีบทวิเสสนะเป็นอุปมา จัดเป็น ๒ ตาม วิเสสนะอยู่หน้าและหลัง.
สมาสที่มีอุปมาอยู่หน้า เรียก "อุปมาปุพฺพปโท" มี อุ. อย่างนี้

| ตัวอย่าง         | สำเร็จรูปเป็น | คำแปล                            |
| ---------------- | ------------- | -------------------------------- |
| สงฺขํ อิว ปณฺฑรํ | สงฺขปณฺฑรํ    | [ ขีรํ น้ำนม ] ขาวเพียงดังสังข์. |
| กาโก อิว สูโร    | กากสูโร       | คนกล้า เพียงดังกา.               |
| ทิพฺพํ อิว จกฺขุ | ทิพฺพจกฺขุ    | จักษุเพียงดังทิพย์.              |

สมาสที่มีอุปมาอยู่หลัง เรียก "อุปมานุตฺตรปโท" มี อุ. อย่างนี้

| ตัวอย่าง         | สำเร็จรูปเป็น | คำแปล                 |
| ---------------- | ------------- | --------------------- |
| นโร สีโห อิว     | นรสีโห        | นระเพียงสีหะ.         |
| ญาณํ จกฺขุ อิว   | ญาณจกฺขุ      | ญาณเพียงดังจักษุ.     |
| ปญฺญา ปาสาโท อิว | ปญฺญาปาสาโท   | ปัญญา เพียงดังปราสาท. |

### สัมภาวนบุพพบท

สัมภาวนบุพพบท มีบทหน้าอันท่านประกอบด้วยอิติศัพท์ บทหลังเป็นประธาน ดังนี้

| ตัวอย่าง                  | สำเร็จรูปเป็น | คำแปล                         |
| ------------------------- | ------------- | ----------------------------- |
| ขตฺติโย [ อหํ ] อิติ มาโน | ขตฺติยมาโน    | มานะว่า [ เราเป็น ] กษัตริย์. |
| สตฺโต อิติ สญฺญา          | สตฺตสญฺญา     | ความสำคัญว่าสัตว์.            |
| สมโณ [ อหํ ] อิติ ปฏิญฺญา | สมณปฏิญฺญา    | ปฏิญญาว่า [ เราเป็น ] สมณะ.   |

### อวธารณบุพพบท

อวธารณบุพพบท มีบทหน้าอันท่านประกอบด้วย เอว ศัพท์ [ เพื่อจะห้ามเนื้อความอันอื่นเสีย] บทหลังเป็นประธาน ดังนี้

| ตัวอย่าง        | สำเร็จรูปเป็น | คำแปล                            |
| --------------- | ------------- | -------------------------------- |
| ปญฺญา เอว ปโชโต | ปญฺญาปโชโต    | [ ประทีป ] อันโพลงทั่ว คือปัญญา. |
| พุทฺโธ เอว รตนํ | พุทฺธรตนํ     | รัตนะ คือพระพุทธเจ้า.            |
| สทฺธา เอว ธนํ   | สทฺธาธนํ      | ทรัพย์คือศรัทธา.                 |

# แบบวิเคราะห์ในคัมภีร์ศัพทศาสตร์

วิธีวางวิเคราะห์แห่งกัมมธารยสมาส ที่เขียนไว้ข้างต้นนั้น มิใช่แบบที่นักปราชญ์ท่านใช้ในคัมภีร์ศัพทศาสตร์ เป็นแต่ข้างเจ้าคิดย่อให้สั้นเข้า จะได้ไม่ต้องแปลวิเคราะห์ให้ยืดยาว แบบวิเคราะห์ที่ท่านใช้ในคัมภีร์ศัพทศาสตร์ทั้งหลาย ดังนี้.

| วิเสสนบุพพบท |                                                          |
| ------------ | -------------------------------------------------------- |
| ตัวอย่าง     | มหนฺโต จ โส ปุริโส จาติ มหาปุริโส.                       |
| คำแปล        | บุรุษนั้นด้วย เป็นผู้ใหญ่ด้วย เหตุนั้น ชื่อบุรุษผู้ใหญ่. |

| วิเสสนุตตรบท |                                                  |
| ------------ | ------------------------------------------------ |
| ตัวอย่าง     | สตฺโต จ โส วิเสโส จาติ สตฺตวิเสโส.               |
| คำแปล        | สัตว์นั้นด้วย วิเศษด้วย เหตุนั้น ชื่อสัตว์วิเศษ. |

| วิเสสโนภยบท |                                                                        |
| ----------- | ---------------------------------------------------------------------- |
| ตัวอย่าง    | สีตญฺจ ตํ สมฏฺฐญฺจาติ สีตตสมฏฺฐํ [ ฐานํ ].                             |
| คำแปล       | [ ที่ ] นั้น เย็นด้วย เกลี้ยงด้วย เหตุนั้น ชื่อว่าทั้งเย็นทั้งเกลี้ยง. |

อีก ๓ สมาสนั้น มีแบบวิเคราะห์เหมือนอย่างในที่นี้.
