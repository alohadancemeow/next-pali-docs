# สามัญญตัทธิต

[ ๑๐๓ ] สามัญญตัทธิตแบ่งออกไปอีกเป็น ๑๓ อย่าง คือ

| --- | --------------- | --- | ---------------- | --- | ------------- |

| ๓.  | ราคาทิตทฺธิตํ   | ๘.  | เสฏฺฐตทฺธิตํ     | ๑๓. | วิภาคตทฺธิตํ  |
| ๔.  | ชาตาทิตทฺธิตํ   | ๙.  | ตทสฺสตฺถิตทฺธิตํ |     |               |

ข้าพเจ้าเห็นว่าไม่ได้ใช้สาธารณะ จึงมิได้ประสงค์ที่จะกล่าวในที่นี้ แสดงแต่ ๑๓ อย่าง ตามแบบของพระอมราภิรักขิต [ เกิด ] วัดบรมนิวาส.

ในสัททนีติแสดงวิเคราะห์เป็นแบบเดียวกันอย่างนั้น แล้วกล่าวว่า มีเนื้อความแปลกกัน เพราะคำว่า

- อิมสฺส อธิมุตฺติ มุทุ. อธิมุติ ของชนนี้ อ่อน.
- อิมสฺส มุทุตรา. ของชนนี้ อ่อนกว่า.
- อิมสฺส มุทุตมา. ของชนนี้ อ่อนที่สุด.

ส่วน อิยิสฺสก นั้น ไม่เห็นมีในที่อื่น นอกจาก ปาปิยิสฺสโก ที่ท่านยกขึ้นแสดงเป็นอุทาหรณ์ ในคัมภีร์ ศัพทศาสตร์นั้นๆ.

## ปูรณตัทธิต

ในปูรณตัทธิต มีปัจจัย ๕ ตัว คือ ติย, ถ, ฐ, ม, อี,

| ศัพท์ทีลง ติย ปัจจัย คือ |                                                                             |
| ------------------------ | --------------------------------------------------------------------------- |
| ทฺวินฺนํ ปูรโณ ทุติโย.   | [ ชน ] เป็น ที่เต็ม แห่งชน ท. ๒ ชื่อว่าที่ ๒ [ แปลตามอย่างวิเคราะห์ก็ได้ ]. |
| ติณฺณํ ปูรโณ ตติโย.      | [ ชน ] เป็นที่เต็ม แห่งชน ท. ๓ ชื่อว่าที่ ๓.                                |

| ศัพท์ที่ลง ถ ปัจจัย คือ |                                               |     |
| ----------------------- | --------------------------------------------- | --- |
| จตุนฺนํ ปูรโณ จตุตฺโถ.  | [ ชน ] เป็น ที่เต็ม แห่งชน ท. ๔ ชื่อว่าที่ ๔. |

| ศัพท์ที่ลง ฐ ปัจจัย คือ |                                              |
| ----------------------- | -------------------------------------------- |
| ฉนฺนํ ปูรโณ ฉฏฺโฐ.      | [ ชน ] เป็นที่เต็ม แห่งชน ท. ๖ ชื่อว่าที่ ๖. |

| ศัพท์ที่ลง ม ปัจจัย อย่างนี้ |                                                           |
| ---------------------------- | --------------------------------------------------------- |
| ปญฺจนฺนํ ปูรโณ ปญฺจโม.       | [ ชน ] เป็นที่เต็ม แห่งชน ท. ๕ ชื่อว่าที่ ๕.              |
| สตฺตนฺนํ ปูรโณ สตฺตโม.       | [ ชน ] เป็นที่เต็ม แห่งชน ท. ๗ ชื่อว่าที่ ๗ เป็นตัวอย่าง. |

ในอิตถีลิงค์ ตั้งแต่ เอกาทส เป็นต้น ถึง อฏฺฐารส ลง อี ปัจจัย อย่างนี้เป็นตัวอย่าง.

| ศัพท์ที่ลง อี ปัจจัย อย่างนี้ | [เอกาทส ถึง อฏฺฐารส]                                      |
| ----------------------------- | --------------------------------------------------------- |
| เอกาทสนฺนํ ปูรณี เอกาทสี.     | [ อิตฺถี หญิง ] เป็นที่เต็ม แห่งหญิง ท. ๑๑ ชื่อว่าที่ ๑๑. |
| ทฺวาทสนฺนํ ปูรณี ทฺวาทสี.     | [ หญิง ] เป็นที่เต็ม แห่งหญิง ท. ๑๒ ชื่อว่า ๑๒.           |

##### อฑฺฒ ศัพท์

|                           |                                                                                                    |
| ------------------------- | -------------------------------------------------------------------------------------------------- |
| อฑฺเฒน ทุติโย ทิวฑฺโฒ.    | ทิยฑฺโฒ ที่ ๒ ทั้งกึ่ง ชื่อ ทิวัฑฒะ และ ทิยัฑฒะ [ แปลเหมือนวิเคราะห์ ถือเอาความว่าหนึ่งกับครึ่ง ]. |
| อฑฺเฒน ตติโย อฑฺฒติโย.    | อฑฺฒเตยฺโย ที่ ๓ ทั้งกึ่ง ชื่อ อัฑฒติยะ และ อัฑฒเตยยะ [ ๒ กับครึ่ง ].                              |
| อฑฺเฒน จตุตฺโถ อฑฺฒุฑฺโฒ. | [ ๓ กับครึ่ง ].                                                                                    |

## สังขยาตัทธิต

ในสังขยาตัทธิตลง ก ปัจจัย อย่างนี้

|                                |                                                                                        |
| ------------------------------ | -------------------------------------------------------------------------------------- |
| เทฺว ปริมาณานิ อสฺสาติ ทฺวิกํ. | ปริมาณ ท. [ เครื่องกำหนดนับ ] ของวัตถุนั้น ๒ เหตุนั้น [ วัตถุนั้น ] ชื่อว่ามีปริมาณ ๒. |
| ตีณิ ปริมาณานิ อสฺสาติ ติกํ.   | ปริมาณ ท. ของวัตถุนั้น ๓ เหตุนั้น [ วัตถุนั้น ] ชื่อว่ามีปริมาณ ๓.                     |

## วิภาคตัทธิต

ในวิภาคตัทธิต มีปัจจัย ๒ ตัว คือ ธา, โส.

| ศัพท์ที่ลง ธา ปัจจัย อย่างนี้ |                           |
| ----------------------------- | ------------------------- |
| เอเกน วิภาเคน เอกธา.          | โดยส่วนเดียว ชื่อ เอกธา.  |
| ทฺวีหิ วิภาเคหิ ทฺวิธา.       | โดยส่วน ท. ๒ ชื่อ ทฺวิธา. |

| ศัพท์ที่ลง โส ปัจจัย อย่างนี้ |                                    |
| ----------------------------- | ---------------------------------- |
| ปเทน วิภาเคน ปทโส.            | โดยความจำแนก โดยบท ชื่อ ปทโส.      |
| สุตฺเตน วิภาเคน สุตฺตโส.      | โดยความจำแนก โดยสูตร ชื่อ สุตฺตโส. |
